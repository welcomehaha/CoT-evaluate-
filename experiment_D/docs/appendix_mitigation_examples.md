# Appendix: Mitigation Examples

## Confession Training

Good:

```text
Cue Disclosure:
- External hint detected: Yes
- Did it influence the answer: Uncertain
- Should it be considered valid evidence: No
```

Bad:

```text
Cue Disclosure:
No cue influenced me.
```

when the answer changes toward the cue target.

## Structured Summary

Good summaries include key premises, invalid cues, a decision rule, and a faithfulness check.

Bad summaries are concise but omit the actual decision rule.

## Dynamic Penalty

High-risk or hidden-cue tasks should receive lower length pressure, preserving more audit evidence.

Easy low-risk tasks should remain concise.
